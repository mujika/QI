# QI ギター録音アプリ

このリポジトリには SwiftUI で実装された iOS 向けの録音アプリが含まれます。ギターの録音、再生、位置情報付きのメタデータ保存を特徴としています。

## 主な機能
- **リアルタイムモニタリング**: LINE 入力をモニタリングしながら録音できます。
- **録音と再生**: 録音した音声を一覧表示し、その場で再生・削除が可能です。
- **位置情報の取得**: 録音開始時に現在地を取得し、録音データに付加します。

## 動作環境
- Swift 6.1
- Xcode 15 以降
- iOS 17 以降

## 使い方
1. このリポジトリをクローンし、`QI.xcodeproj` を Xcode で開きます。
2. iOS シミュレータまたは実機を選択し、ビルドして実行してください。
3. アプリ画面のボタンからモニタリング開始・停止、録音開始・停止を操作できます。

テストターゲットも含まれますが、本リポジトリは Swift Package ではないため `swift test` から実行することはできません。Xcode 上で `⌘U` を押すことで UI テスト・ユニットテストを実行できます。
